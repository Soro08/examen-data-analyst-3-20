# examen-data-analyst-3-20


Après vôtre certification NaN 2020, vous postulé pour le post de Data Analyst.
Pendant le recrutement, le responsable du recrutement demande à chaque canditat de faire preuve de créativité.

On vous demande alors de choisir une dataset, éffectué une étude complète sur cette data et expliquer le travail que vous allez effectuer.


Durée : 7h


NB: avant la fin de l'épreuve, chaque candidat doit envoyer son fichier d'exercice dans laquelle ils expliquera:

- Le choix le choix du dataset
- Le travail éffectué
- Les methodes utilisé
